# 古今异义词汇知识库

## 概述

汉语词义演变所产生的古今异义现象，不仅是汉语史研究和古籍整理注释关注的重点，也是语言学习面临的难点。为了全面地梳理古今异义现象，挖掘古今汉语的词汇语义关联，本研究**基于大语言模型构建了义项粒度的古今异义自动分析方法**，并在此基础上构建了**大规模古今异义词汇知识库**。

具体而言，我们首先设计了词语义项的古今异义判别与理解难度标注原则，并据此构建了人工标注数据集；其次提出了基于义频统计和义项匹配的两种自动标注方法，对义项是否属于古今异义进行判定，实验结果显示，自动标注方法的准确率可达到 85% 以上；进一步地，我们对大规模词义数据进行了自动标注和人工复核，最终构建了一个包含 **2529 个词（2405 个简体词形）、24824 个义项**的古今异义词汇知识库，其中有 14258 个义项被标记为古今异义义项,以期为语言教学和学术研究提供资源参考。

## 古今异义词汇知识库基本结构

### 字段说明

| 字段名            | 类型    | 说明                                             |
|----------------|-------|------------------------------------------------|
| `ID`           | 整数    | 目标义项的唯一数字标识                                    |
| `token_simp`   | 字符串   | 以简体字形记录的目标词                                    |
| `token_trad`   | 字符串   | 以繁体字形记录的目标词                                    |
| `sense`        | 字符串   | 目标词在《汉语大词典》中的义项条目                              |
| `is_diff`      | 布尔值   | 该义项是否属于古今异义义项（`true` 表示是，`false` 表示否）          |
| `modern_match` | 字符串   | 目标义项在《现代汉语词典》（第 7 版）中的关联义项；若无匹配项，则为 `无匹配项`    |
| `modern_simi`  | 浮点数   | 目标义项与关联现汉义项之间的语义相似度（取值范围 0–1）                  |
| `sent`         | 字符串   | 目标义项在文献中的例句                                    |

### 知识库结构设计说明

- **词语收录**：依托「AI 太炎」自动标注的大规模古汉语历时词义语料库（详见[古汉语历时词义语料库] https://corpus.shenshen.wiki ）所提供的义频信息，选取了 2529 个词义变化度较高的词语作为潜在的古今异义词，进行自动标注与分析。
- **义项收录**：主要参考《汉语大词典》的义项收录情况。《汉语大词典》以「古今兼收，源流并重」为编辑方针，基于该辞书的义项来构建知识库，能够较好地兼顾词语的古义与今义，满足古今义项差异分析的需要。
- **知识库属性设置**：
  - **词语形式**：考虑到一简对多繁的情况，同时记录了词语的简体和繁体形式，以满足不同的查询需求。
  - **关联现汉义项与相似度信息**：为了更好地揭示古今词义之间的联系，我们列出了目标义项在《现代汉语词典》中的关联义项，并计算了二者之间的语义相似度。实验发现，相似度大于 0.75 的匹配义项可视为语义十分相近的匹配项；相似度小于 0.75 的则可能存在语义不完全相近的情况，使用时建议再次核对。
  - **例句**：为帮助用户更好地理解义项的语义内涵，我们收录了目标义项在文献中的例句。需要说明的是，例句主要来源于《汉语大词典》，部分生僻义项可能暂缺例句。

### 数据形式

为满足学术研究和教学使用的不同需求，我们提供三种数据格式的知识库版本：

- **JSONL 格式**：每行一个 JSON 对象，适合学术研究场景下的批量数据处理与分析。
- **XLSX 格式**：以表格形式呈现，便于教师与学生浏览和查询，适合教学使用。
- **DOCX 格式**：该版本仅收录**属于古今异义义项**的条目，便于在文言文教学与古籍阅读中重点参考。

## 反馈与迭代

本知识库在自动标注与人工复核相结合的基础上构建，仍可能存在标注不完善之处。欢迎使用者通过 GitHub Issue 或 Pull Request 提出反馈与建议，我们将持续迭代，不断提升数据质量。
